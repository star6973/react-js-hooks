# react-js-hooks
실전형 리액트 Hooks

리액트 훅은 functional component에서 state를 가질 수 있게 한다.
가장 많이 사용하고 있는 훅인 useState는 2가지를 반환한다.
첫 번째 value는 item이라는 이름을 주고, 두 번째 value는 item 값을 변경할 수 있는 set 함수이다.
